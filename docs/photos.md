# 图库设置 (Photos)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 图库**

需安装 `plugin-photos` 插件。

---

## 页面设置

| 设置项           | 说明                                                                                                                                   | 默认值                 |
| ---------------- | -------------------------------------------------------------------------------------------------------------------------------------- | ---------------------- |
| **页面标题**     | 图库页面的标题                                                                                                                         | "图库"                 |
| **页面描述**     | 图库页面的描述文字                                                                                                                     | "记录生活中的美好瞬间" |
| **显示图片信息** | 图片下方/悬浮时显示标题和描述                                                                                                          | 关闭                   |
| **显示下载按钮** | 悬浮遮罩中显示原图下载按钮（与图片信息解耦控制）；照片详情页也会使用该开关控制「下载原图」按钮                                         | 关闭                   |
| **启用评论**     | 在图库页面底部显示评论区。**注意**：为确保评论区标题显示正确（而非"未知"），请在 Halo 后台新建一个别名(Slug)为 `photos` 的自定义页面。 | 开启                   |

## 布局外观 (New)

图库底层采用高级 JS Masonry 物理分排算法（彻底告别原生 CSS Columns 瀑布流排版错位的通病），并配合智能无限滚动加载。您现在可以通过后台直观地控制其外观：

| 设置项       | 说明                                                          |
| ------------ | ------------------------------------------------------------- |
| **页面布局** | `居中式`（经典等宽阅读区）或 `全铺式`（边缘小留白，沉浸全屏） |
| **最大列数** | 控制宽屏下最多显示几列图片（支持 2 ~ 6 列选择）               |
| **图片圆角** | 自定义图片边角弧度，支持数字输入（建议 0px ~ 24px）           |
| **图片间距** | 自定义图片之间的缝隙大小，支持数字输入（建议 0px ~ 32px）     |

## 照片详情页

主题提供 `photo.html` 覆盖 `plugin-photos` 的照片详情路由。详情页会使用插件传入的 `photo`、`prev`、`next`、`neighbors`、`photoUrl` 和 `exif` 变量渲染：

| 区域          | 说明                                                          |
| ------------- | ------------------------------------------------------------- |
| **照片舞台**  | 使用原图等比展示，支持上一张 / 下一张导航                     |
| **信息面板**  | 展示标题、描述、标签和原图操作                                |
| **EXIF 参数** | 有数据时展示相机、镜头、光圈、快门、ISO、焦距、拍摄时间和尺寸 |
| **附近探索**  | 使用 `neighbors` 渲染同组附近照片                             |

如果插件没有提供 `photoUrl`，主题会隐藏上下张和附近照片链接，避免详情页模板报错。

## PJAX 与灯箱

图库脚本跟随主题 `SkyPjax` 生命周期初始化：

- PJAX 进入 `/photos` 时重新初始化 Masonry、详情按钮拦截和无限滚动。
- PJAX 离开图库时清理 resize 监听、图片加载监听、预加载 timeout 和 `IntersectionObserver`。
- 点击图片本身仍交给 lightgallery 插件处理；点击「详情」按钮才进入照片详情页。
- lightgallery 插件的图库匹配区域建议只配置 `/photos` + `#photo-grid`，详情页不需要配置 `#photo-grid`。

## 悬浮控制栏

| 设置项             | 说明                                 | 默认值  |
| ------------------ | ------------------------------------ | ------- |
| **定制悬浮控制栏** | 启用后可单独配置，禁用则使用全局配置 | 关闭    |
| **Dock 菜单**      | 选择要在悬浮控制栏中显示的菜单       | primary |
| **返回顶部按钮**   | 是否显示返回顶部按钮                 | 开启    |
