# 归档页设置 (Archive)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 归档**

---

## 侧边栏设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **显示侧边栏** | 控制归档页是否显示侧边栏 | 关闭 |
| **定制侧边栏** | 启用后可单独配置，禁用则使用全局配置 | 关闭 |
| **侧边栏位置** | 左侧 / 右侧 | 右侧 |
| **小工具列表** | 拖拽排序添加小工具 | - |

### 可用小工具
欢迎卡片（天气）、作者卡片、博客统计、文章分类、标签云、热门文章、最新文章、**归档探索**

> **💡 新增特性：归档探索 (Global Explorer)**
> 主题现已内置专属的 `归档探索` 侧边栏小工具。它通过全局 `postFinder` 抓取所有年份和月份数据（而非仅显示当前分页数据），并以精致的扁平化无框排版呈现树状目录。点击月份将使用原生路由 (`/archives/YYYY/MM`) 进行精准过滤浏览，小工具图标支持在附加设定中进行 Iconify 图标自定义。

---

## 交互功能

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **启用评论** | 在归档页面底部显示评论区。**注意**：为确保评论区标题显示正确（而非"未知"），请在 Halo 后台新建一个别名(Slug)为 `archives` 的自定义页面。 | 关闭 |

---

## 页面视觉与排版

首页归档大标题采用了定制的优雅衬线体 (`font-serif`) 排印，并且副标题的`总年份统计`不再受分页限制，现已升级为**全局年份总数**统计，为您呈现更准确的记录概览。

---

## 悬浮控制栏

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **定制悬浮控制栏** | 启用后可单独配置 | 关闭 |
| **Dock 菜单** | 选择菜单 | primary |
| **返回顶部按钮** | 是否显示 | 开启 |
