# 顶部导航设置 (Nav)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 顶部（全局）**

配置网站顶部导航栏的外观和功能。

---

## 1. Logo 设置

| 设置项 | 说明 |
|--------|------|
| **Logo 图标类型** | `隐藏` / `设置-Logo` / `Iconify` / `SVG` / `上传图片` |
| **Logo 图标** | 使用 Iconify 图标库选择图标 |
| **Logo SVG 代码** | 直接输入 SVG 代码渲染 |
| **Logo 图片** | 上传自定义 Logo 图片（建议 32x32 或 64x64） |
| **显示 Logo 标题** | 是否显示 Logo 右侧的标题文字 |
| **自定义 Logo 标题** | 留空使用站点标题，填写则使用自定义标题 |

---

## 2. 菜单设置

| 设置项 | 说明 |
|--------|------|
| **导航栏菜单** | 选择要在主导航栏显示的菜单 |

> **💡 提示**：在 Halo 后台的 **菜单管理** 中配置菜单项。支持多级下拉菜单。

---

## 3. 主题设置

配置明暗主题切换和主题配色。

| 设置项 | 说明 |
|--------|------|
| **主题前端开关** | 是否显示右上角的主题切换按钮 |
| **浅色主题图标** | 自定义浅色主题按钮图标 |
| **深色主题图标** | 自定义深色主题按钮图标 |
| **默认主题** | `浅色主题` 或 `深色主题` |
| **浅色主题** | 选择浅色模式配色方案 |
| **深色主题** | 选择深色模式配色方案 |

### 支持的浅色主题
Light, Cupcake, Bumblebee, Emerald, Corporate, Retro, Cyberpunk, Valentine, Garden, Lofi, Pastel, Fantasy, Wireframe, CMYK, Autumn, Acid, Lemonade, Winter, Nord, Caramellatte, Silk

### 支持的深色主题
Dark, Synthwave, Halloween, Forest, Aqua, Black, Luxury, Dracula, Business, Night, Coffee, Dim, Sunset, Abyss

> **💡 说明**：主题配色基于 [DaisyUI](https://daisyui.com/theme-generator/)，可访问官网查看具体配色方案。

---

## 4. 功能设置

配置导航栏右侧的功能按钮。

| 设置项 | 说明 |
|--------|------|
| **搜索按钮** | 是否显示搜索按钮（需安装搜索插件） |
| **搜索按钮图标** | 自定义搜索按钮图标 |
| **登录按钮** | 是否显示登录/用户菜单按钮 |
| **登录按钮图标** | 自定义登录按钮图标 |

> **💡 提示**：搜索功能需安装 `PluginSearchWidget` 插件才能使用。主题已为搜索组件适配了**极简列表风格 (Command Palette)**，提供更沉浸的搜索体验。
